ProgramaEncuestaRenew
Fecha: 05/05/2025
Autor: mauro.av

Descripción
ProgramaEncuestaRenew tiene como finalidad automatizar la encuesta de evaluación docente utilizando Python y Selenium, logrando así disminuir los tiempos de evaluación.

Características de la nueva versión
Implementación de ventanas gráficas para facilitar la interacción del usuario.
Recopilación automática de los códigos de materia, eliminando la necesidad de ingresarlos manualmente.
Una vez obtenidos los códigos, se mostrará un mensaje de confirmación para que el usuario valide si corresponden a las materias que está cursando.

Proceso de Evaluación
Se desplegará un menú con estrellas para calificar cada materia.
Se incluirá un campo de texto para que el usuario brinde su opinión sobre su experiencia.
La media de estrellas será utilizada como la calificación del docente.
El texto ingresado será utilizado como retroalimentación en la caja de comentarios.
